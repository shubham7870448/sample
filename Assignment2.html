Difference Between

HTML TAGS and ATTRIBUTES:

TAGS: ATTRIBUTES:
  1. They contain elements which define how the information on a web page is formatted or displayed.
  2. HTML Tags is enclosed between < and >
  3. HTML tags doesn't include quotes.
  4. e.g, HR, ol , a , etc.
  1. Attribute is a property that provides additional information about an HTML element
  2. An attribute is always specified inside the opening tag.
  3. Attributes value is always enclosed within quotes.
  4. e.g, height , href , src , etc.
